## 1. Introduction to AngularJS

### What is AngularJS?
AngularJS is a structural framework for dynamic web applications. It extends HTML with additional attributes and binds data to HTML with expressions.

### Role of AngularJS in Web Development
- **Single Page Applications (SPAs)**: Creates dynamic, responsive web applications
- **MVC Architecture**: Separates concerns between Model, View, and Controller
- **Data Binding**: Automatically synchronizes data between model and view
- **Dependency Injection**: Makes applications more modular and testable
- **Templating**: Extends HTML with custom elements and attributes

### Key Benefits
- Reduces code complexity
- Improved code organization
- Enhanced testing capabilities
- Rich user interface components
- Cross-browser compatibility

## 2. Basic Architecture and Core Concepts

### MVC/MVVM Pattern
- **Model**: Data and business logic
- **View**: User interface (HTML templates)
- **Controller**: Handles user interactions and updates the model
- **ViewModel**: Binds view and model together

### Core Concepts

#### Modules
Containers for different parts of your application (controllers, services, directives, etc.)

#### Controllers
JavaScript functions that augment the Angular Scope

#### Scope
Binding between HTML (view) and JavaScript (controller)

#### Services
Reusable business logic independent of views

#### Directives
Markers in HTML that tell AngularJS to attach specific behavior

#### Filters
Format data for display to users

## 3. Creating Your First AngularJS Application

### Step 1: Set Up HTML Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First AngularJS App</title>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.9/angular.min.js"></script>
</head>
<body ng-app="myApp" ng-controller="myController">
    <h1>Welcome to AngularJS!</h1>
    <p>Hello, {{name}}!</p>
    <input type="text" ng-model="name" placeholder="Enter your name">
    
    <script src="app.js"></script>
</body>
</html>
```

### Step 2: Create the JavaScript Application (app.js)

```javascript
// Create the module
var app = angular.module('myApp', []);

// Create the controller
app.controller('myController', function($scope) {
    $scope.name = 'World';
});
```

### Step 3: Understanding the Code
- `ng-app="myApp"`: Defines the AngularJS application
- `ng-controller="myController"`: Defines which controller handles this section
- `{{name}}`: Expression that displays the value of name
- `ng-model="name"`: Two-way data binding for the input field

## 4. AngularJS Directives

### Built-in Directives

#### Basic Directives

**ng-show / ng-hide**
```html
<div ng-show="showElement">This will show when showElement is true</div>
<div ng-hide="hideElement">This will hide when hideElement is true</div>
```

**ng-if**
```html
<div ng-if="condition">This element is conditionally rendered</div>
```

**ng-repeat**
```html
<ul>
    <li ng-repeat="item in items">{{item.name}}</li>
</ul>
```

**ng-class**
```html
<div ng-class="{'active': isActive, 'disabled': isDisabled}">Dynamic classes</div>
```

#### Advanced Directives

**ng-switch**
```html
<div ng-switch="selection">
    <div ng-switch-when="option1">Option 1 selected</div>
    <div ng-switch-when="option2">Option 2 selected</div>
    <div ng-switch-default>Default option</div>
</div>
```

### Custom Directives

#### Creating a Simple Custom Directive

```javascript
app.directive('myCustomDirective', function() {
    return {
        template: '<div>This is a custom directive!</div>'
    };
});
```

#### More Advanced Custom Directive

```javascript
app.directive('userCard', function() {
    return {
        restrict: 'E', // Element directive
        scope: {
            user: '=' // Isolated scope
        },
        template: `
            <div class="user-card">
                <h3>{{user.name}}</h3>
                <p>{{user.email}}</p>
                <p>{{user.role}}</p>
            </div>
        `,
        link: function(scope, element, attrs) {
            // DOM manipulation logic here
            element.on('click', function() {
                alert('User card clicked: ' + scope.user.name);
            });
        }
    };
});
```

Usage:
```html
<user-card user="currentUser"></user-card>
```

## 5. Two-Way Data Binding

### Understanding Data Binding

Two-way data binding automatically synchronizes data between the model and view layers.

### Examples

#### Basic Two-Way Binding
```html
<div ng-controller="bindingController">
    <input type="text" ng-model="message">
    <p>You typed: {{message}}</p>
</div>
```

```javascript
app.controller('bindingController', function($scope) {
    $scope.message = 'Hello, AngularJS!';
});
```

#### Complex Object Binding
```html
<div ng-controller="userController">
    <input type="text" ng-model="user.firstName" placeholder="First Name">
    <input type="text" ng-model="user.lastName" placeholder="Last Name">
    <input type="email" ng-model="user.email" placeholder="Email">
    
    <h3>User Information:</h3>
    <p>Name: {{user.firstName}} {{user.lastName}}</p>
    <p>Email: {{user.email}}</p>
</div>
```

```javascript
app.controller('userController', function($scope) {
    $scope.user = {
        firstName: '',
        lastName: '',
        email: ''
    };
});
```

## 6. Filters for Data Formatting

### Built-in Filters

#### Currency Filter
```html
<p>Price: {{product.price | currency}}</p>
<p>Price in EUR: {{product.price | currency:'â‚¬'}}</p>
```

#### Date Filter
```html
<p>Current Date: {{currentDate | date}}</p>
<p>Custom Format: {{currentDate | date:'dd/MM/yyyy'}}</p>
<p>Time: {{currentDate | date:'HH:mm:ss'}}</p>
```

#### Uppercase/Lowercase Filters
```html
<p>{{name | uppercase}}</p>
<p>{{name | lowercase}}</p>
```

#### Number Filter
```html
<p>{{number | number:2}}</p> <!-- 2 decimal places -->
```

#### Filter Filter (for arrays)
```html
<ul>
    <li ng-repeat="item in items | filter:searchText">{{item.name}}</li>
</ul>
<input type="text" ng-model="searchText" placeholder="Search...">
```

#### OrderBy Filter
```html
<ul>
    <li ng-repeat="user in users | orderBy:'name'">{{user.name}} - {{user.age}}</li>
</ul>
<ul>
    <li ng-repeat="user in users | orderBy:'-age'">{{user.name}} - {{user.age}}</li>
</ul>
```

### Custom Filters

#### Creating a Custom Filter
```javascript
app.filter('reverse', function() {
    return function(input) {
        if (!input) return '';
        return input.split('').reverse().join('');
    };
});

// Usage in HTML
// <p>{{message | reverse}}</p>
```

#### Advanced Custom Filter
```javascript
app.filter('capitalize', function() {
    return function(input) {
        if (!input) return '';
        return input.charAt(0).toUpperCase() + input.slice(1).toLowerCase();
    };
});
```

## 7. Controllers

### Basic Controller Structure

```javascript
app.controller('mainController', function($scope) {
    // Controller logic here
    $scope.title = 'My Application';
    $scope.users = [
        {name: 'John', age: 25},
        {name: 'Jane', age: 30},
        {name: 'Bob', age: 35}
    ];
    
    $scope.addUser = function() {
        $scope.users.push({
            name: $scope.newUserName,
            age: $scope.newUserAge
        });
        // Clear form
        $scope.newUserName = '';
        $scope.newUserAge = '';
    };
});
```

### Controller with Services

```javascript
// Service
app.service('userService', function() {
    var users = [];
    
    this.getUsers = function() {
        return users;
    };
    
    this.addUser = function(user) {
        users.push(user);
    };
    
    this.removeUser = function(index) {
        users.splice(index, 1);
    };
});

// Controller using the service
app.controller('userController', function($scope, userService) {
    $scope.users = userService.getUsers();
    
    $scope.addUser = function(user) {
        userService.addUser(user);
    };
    
    $scope.removeUser = function(index) {
        userService.removeUser(index);
    };
});
```

### Controller Best Practices

#### Using Controller As Syntax
```javascript
app.controller('betterController', function() {
    var vm = this; // ViewModel pattern
    
    vm.title = 'Better Controller';
    vm.users = [];
    
    vm.addUser = function(user) {
        vm.users.push(angular.copy(user));
        vm.newUser = {}; // Clear form
    };
});
```

HTML usage:
```html
<div ng-controller="betterController as ctrl">
    <h1>{{ctrl.title}}</h1>
    <ul>
        <li ng-repeat="user in ctrl.users">{{user.name}}</li>
    </ul>
</div>
```

## 8. Angular Forms and Events

### Basic Form Structure

```html
<form name="userForm" ng-submit="submitForm()" novalidate>
    <div>
        <label>Name:</label>
        <input type="text" name="name" ng-model="user.name" required>
        <div ng-show="userForm.name.$invalid && userForm.name.$touched">
            Name is required
        </div>
    </div>
    
    <div>
        <label>Email:</label>
        <input type="email" name="email" ng-model="user.email" required>
        <div ng-show="userForm.email.$invalid && userForm.email.$touched">
            Valid email is required
        </div>
    </div>
    
    <div>
        <label>Age:</label>
        <input type="number" name="age" ng-model="user.age" min="18" max="120">
        <div ng-show="userForm.age.$invalid && userForm.age.$touched">
            Age must be between 18 and 120
        </div>
    </div>
    
    <button type="submit" ng-disabled="userForm.$invalid">Submit</button>
</form>
```

### Event Handling

#### Common Events

**ng-click**
```html
<button ng-click="handleClick()">Click Me</button>
<button ng-click="counter = counter + 1">Count: {{counter}}</button>
```

**ng-submit**
```html
<form ng-submit="submitForm(userForm.$valid)" novalidate>
    <!-- form content -->
</form>
```

**ng-change**
```html
<select ng-model="selectedOption" ng-change="handleChange()">
    <option value="option1">Option 1</option>
    <option value="option2">Option 2</option>
</select>
```

**ng-blur / ng-focus**
```html
<input type="text" ng-focus="onFocus()" ng-blur="onBlur()" ng-model="inputValue">
```

#### Event Handler Examples

```javascript
app.controller('eventController', function($scope) {
    $scope.counter = 0;
    
    $scope.handleClick = function() {
        alert('Button clicked!');
    };
    
    $scope.handleChange = function() {
        console.log('Selection changed to:', $scope.selectedOption);
    };
    
    $scope.onFocus = function() {
        console.log('Input focused');
    };
    
    $scope.onBlur = function() {
        console.log('Input lost focus');
    };
    
    $scope.submitForm = function(isValid) {
        if (isValid) {
            console.log('Form submitted:', $scope.user);
        } else {
            alert('Please fix form errors');
        }
    };
});
```

## 9. Form Validation

### Built-in Validation

#### Required Fields
```html
<input type="text" name="username" ng-model="user.username" required>
<div ng-show="myForm.username.$error.required && myForm.username.$touched">
    Username is required
</div>
```

#### Email Validation
```html
<input type="email" name="email" ng-model="user.email" required>
<div ng-show="myForm.email.$error.email && myForm.email.$touched">
    Please enter a valid email
</div>
```

#### Pattern Validation
```html
<input type="text" name="phone" ng-model="user.phone" 
       ng-pattern="/^[0-9]{10}$/" required>
<div ng-show="myForm.phone.$error.pattern && myForm.phone.$touched">
    Phone number must be 10 digits
</div>
```

#### Min/Max Length
```html
<input type="password" name="password" ng-model="user.password" 
       ng-minlength="8" ng-maxlength="20" required>
<div ng-show="myForm.password.$error.minlength">
    Password must be at least 8 characters
</div>
<div ng-show="myForm.password.$error.maxlength">
    Password cannot exceed 20 characters
</div>
```

### Advanced Validation Example

```html
<form name="registrationForm" ng-submit="register()" novalidate>
    <div class="form-group">
        <label>Username:</label>
        <input type="text" name="username" ng-model="user.username" 
               required ng-minlength="3" ng-maxlength="20">
        
        <div class="error-messages">
            <div ng-show="registrationForm.username.$error.required && registrationForm.username.$touched">
                Username is required
            </div>
            <div ng-show="registrationForm.username.$error.minlength">
                Username must be at least 3 characters
            </div>
            <div ng-show="registrationForm.username.$error.maxlength">
                Username cannot exceed 20 characters
            </div>
        </div>
    </div>
    
    <div class="form-group">
        <label>Email:</label>
        <input type="email" name="email" ng-model="user.email" required>
        
        <div class="error-messages">
            <div ng-show="registrationForm.email.$error.required && registrationForm.email.$touched">
                Email is required
            </div>
            <div ng-show="registrationForm.email.$error.email">
                Please enter a valid email address
            </div>
        </div>
    </div>
    
    <div class="form-group">
        <label>Password:</label>
        <input type="password" name="password" ng-model="user.password" 
               required ng-minlength="8">
        
        <div class="error-messages">
            <div ng-show="registrationForm.password.$error.required && registrationForm.password.$touched">
                Password is required
            </div>
            <div ng-show="registrationForm.password.$error.minlength">
                Password must be at least 8 characters
            </div>
        </div>
    </div>
    
    <div class="form-group">
        <label>Confirm Password:</label>
        <input type="password" name="confirmPassword" ng-model="user.confirmPassword" 
               required compare-to="user.password">
        
        <div class="error-messages">
            <div ng-show="registrationForm.confirmPassword.$error.required && registrationForm.confirmPassword.$touched">
                Please confirm your password
            </div>
            <div ng-show="registrationForm.confirmPassword.$error.compareTo">
                Passwords do not match
            </div>
        </div>
    </div>
    
    <button type="submit" ng-disabled="registrationForm.$invalid">
        Register
    </button>
    
    <div ng-show="registrationForm.$invalid && registrationForm.$submitted">
        Please fix the errors above
    </div>
</form>
```

### Custom Validation Directive

```javascript
app.directive('compareTo', function() {
    return {
        require: 'ngModel',
        scope: {
            otherModelValue: '=compareTo'
        },
        link: function(scope, element, attributes, ngModel) {
            ngModel.$validators.compareTo = function(modelValue) {
                return modelValue === scope.otherModelValue;
            };
            
            scope.$watch('otherModelValue', function() {
                ngModel.$validate();
            });
        }
    };
});
```

## 10. Debugging AngularJS Applications

### Using Browser Developer Tools

#### Console Debugging

**Access Scope in Console:**
```javascript
// Select an element and access its scope
var element = document.querySelector('[ng-controller="myController"]');
var scope = angular.element(element).scope();
console.log(scope);
```

**Inspect Scope Variables:**
```javascript
// In controller, add console.log statements
app.controller('debugController', function($scope) {
    $scope.data = 'Initial value';
    
    $scope.updateData = function() {
        console.log('Before update:', $scope.data);
        $scope.data = 'Updated value';
        console.log('After update:', $scope.data);
    };
});
```

#### Angular Inspector

**Enable Debug Info:**
```javascript
// Enable debug info (enabled by default in development)
app.config(function($compileProvider) {
    $compileProvider.debugInfoEnabled(true);
});
```

### Common Debugging Techniques

#### 1. Using {{}} for Quick Debugging
```html
<div>
    <p>Debug: {{$ctrl.debugVariable | json}}</p>
    <p>Form valid: {{myForm.$valid}}</p>
    <p>Form errors: {{myForm.$error | json}}</p>
</div>
```

#### 2. Logging in Controllers
```javascript
app.controller('debugController', function($scope, $log) {
    $log.debug('Controller initialized');
    
    $scope.items = [];
    
    $scope.addItem = function(item) {
        $log.info('Adding item:', item);
        $scope.items.push(item);
        $log.debug('Current items:', $scope.items);
    };
});
```

#### 3. Using $watch for Monitoring Changes
```javascript
app.controller('watchController', function($scope) {
    $scope.watchedVariable = '';
    
    // Watch for changes
    $scope.$watch('watchedVariable', function(newValue, oldValue) {
        console.log('Variable changed from', oldValue, 'to', newValue);
    });
    
    // Watch collection changes
    $scope.$watchCollection('items', function(newItems, oldItems) {
        console.log('Items collection changed');
    });
});
```

#### 4. Exception Handling
```javascript
app.config(function($provide) {
    $provide.decorator('$exceptionHandler', function($delegate, $log) {
        return function(exception, cause) {
            $log.error('Application error:', exception);
            console.trace();
            $delegate(exception, cause);
        };
    });
});
```

### Debugging Tools and Extensions

#### Browser Extensions
- **AngularJS Batarang** (Chrome extension)

**$timeout for Debugging Async Issues:**

```javascript
app.controller('asyncController', function($scope, $timeout) {

            // Simulate API call

```

### Performance Debugging

#### Identifying Digest Cycle Issues
```javascript
app.run(function($rootScope) {
    var watchers = 0;
    
    $rootScope.$watch(function() {
        watchers++;
        if (watchers % 100 === 0) {
            console.log('Digest cycle count:', watchers);
        }
    });
});
```

#### Optimizing ng-repeat
```html
<!-- Use track by for better performance -->
<li ng-repeat="item in items track by item.id">{{item.name}}</li>

<!-- Instead of -->
<li ng-repeat="item in items">{{item.name}}</li>
```
