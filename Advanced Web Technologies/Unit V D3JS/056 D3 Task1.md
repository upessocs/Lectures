Background Notes (Part of the Question)

D3.js (Data-Driven Documents) is a JavaScript library that allows you to connect data with DOM elements.
The core mechanism is:

d3.select("#container")
  .selectAll("li")
  .data(data);

When data is bound, D3 logically divides the selection into:

1. enter — items in the new data that do not yet have DOM elements


2. update — items that match existing DOM elements


3. exit — DOM elements that no longer have matching data



Traditionally, developers used the enter–update–exit pattern:

const sel = d3.selectAll("li").data(arr);

sel.enter().append("li");
sel.text(d => d);
sel.exit().remove();

Modern D3 versions provide the .join() method, which combines all three steps:

d3.select("#list")
  .selectAll("li")
  .data(arr)
  .join("li")              // handles enter, update, exit
  .text(d => d);

.join("li") automatically:

Appends <li> for enter items

Reuses existing nodes for update

Removes exit nodes
This gives a shorter and safer update pattern.
